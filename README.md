# Inquisition 审判庭

明日方舟速通云控后端

## 什么是 Inquisition 审判庭

Inquisition 是一款基于 [ArkLights](https://github.com/tkkcc/ArkLights)
开发的云控后端程序，是集成了多账号多终端负载均衡，数据管理，任务分发，消息推送，日志等等功能于一体的强大后端管理程序，使用
Inquisition 可以轻松管理数百个账号与设备。
